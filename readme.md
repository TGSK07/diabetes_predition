# 🩺 Diabetes Prediction using ML & Deep Learning

Hey there! 👋  
This project is all about predicting whether a person is likely to have diabetes using Machine Learning and Deep Learning models.

Diabetes is one of the most common chronic diseases worldwide. Early prediction can help people take preventive steps, improve lifestyle, and avoid serious health complications.

This project tries to solve that problem by using data-driven models to make predictions quickly and efficiently.

---

## 🧠 Models Used

I experimented with multiple models to compare performance:

- 🔹 K-Nearest Neighbors (KNN)  
- 🌳 Decision Tree  
- 🧠 MLP (Multi-Layer Perceptron)  
- 🤖 Deep Learning Model (Neural Network using Keras)

---

## 📊 Evaluation Metrics

To evaluate how well the models perform, I used:

- ✅ Accuracy  
- 🔁 Recall  
- 🎯 F1-Score  
- 📈 ROC-AUC Score  

These metrics help understand not just correctness, but also how well the model handles imbalanced data.

---

## 🔍 Feature Importance

Some features play a major role in predicting diabetes:

- Glucose 🧪 (most important!)  
- BMI ⚖️  
- Age 👤  
- Blood Pressure 💉  
- Insulin 💊  

Understanding these features helps make the model more interpretable and meaningful.
